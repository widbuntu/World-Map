About This Application

Purpose

This application provides a world map that allows users to hover over countries and get detailed information about specific countries. The data includes country name, region, income level, capital city, lending type, and employment in agriculture.

Sources

Map: https://simplemaps.com/static/demos/resources/svg-library/svgs/world.svg - A reliable SVG map resource.
Data: https://api.worldbank.org/v2/country - Data sourced from the World Bank API.
