About This Application

This application provides an interactive world map where you can hover over countries to get detailed information about them. The data includes country name, region, income level, capital city, lending type, and employment in agriculture.

Features

  • Interactive map.
  • Data fetched from the World Bank API.
  • Detailed information about countries, including economic indicators.

How to Use

  1. Navigate to the Map page.
  2. Hover over a country to see detailed information about it.
  3. Navigate to the Sources page for more information about the application.
